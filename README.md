## Highlights
- Movie details with synopsis, runtime, genres, and TMDB ratings
- Watch rooms to plan sessions with friends
- Reviews and ratings out of 10
- Favorites and personal lists
- Admin panel to manage users and permissions

## Open Source and Self-Hosted
Cineclub is fully open source and built to run on your own server. That means:
- Full data ownership
- No third-party dependency
- Transparent code that you can audit and improve

## Security and Administration
Cineclub includes modern security and a complete admin dashboard:
- Secure authentication and encryption
- Granular user management
- Activity and audit visibility

## Community
Cineclub is a community-driven project. Ideas, bug reports, and contributions are welcome.
